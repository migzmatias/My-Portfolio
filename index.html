Simple Task Management Application

This Task Management Application built using C# is a console-based system that allows users to create, assign, track, and verify tasks while storing data in an external CSV file. It records task details such as description, timestamps, assigned person, and status updates, ensuring efficient task monitoring. The application supports different task statuses (Open, Assigned, For Verification, For Revision, Closed) and includes a verification to validate completed tasks. The system ensures data integrity and provides a user- way to manage and track tasks effectively.

8-Bit Binary Conversion Game

This game was developed using WPF and C#. The goal is to convert as much binary numbers you can into decimal numbers under time pressure. The game features a scoring system that tracks the player's progress, a leaderboard that records high scores, and an input system for entering player names. A leaderboard window sorts and displays top scores based on performance, time played, and difficulty selected making for a simple yet engaging gameplay experience.

Sleep Health and Lifetyle Dataset Analysis

This dashboard was created using Microsoft's PowerBI. It is an analysis on a dataset named "Sleep Health and Lifestyle" taken from the website "Kaggle". It contains information on various health and lifestyle factors such as sleep duration, stress levels, physical activity, blood pressure, and more. This dataset was used to explore the correlations and trends in health metrics, aiming to identify actionable insights. This analysis and visualization on the dataset “Sleep Health and Lifestyle” provides comprehensive insights into the relationships between the aforementioned health and lifestyle factors separated by demographic groups such as gender and age.
